# Professional Theme

Clean and professional dark theme with blue accents.

## Required Assets

### banner.jpg
Place your banner image in this directory as `banner.jpg`.

**Recommended specifications:**
- **Dimensions**: 1920x400px (or similar wide aspect ratio)
- **Format**: JPG or PNG
- **Style**: Professional, corporate imagery (abstract tech, circuits, data visualization, etc.)
- **Color palette**: Blues, grays, cool tones to match the theme

## Theme Colors

- Primary: Blue (#2563eb)
- Accent: Cyan (#0ea5e9)
- Background: Dark slate (#0f172a)
- Text: Light gray/white

## Customization

Edit `theme.json` to modify colors, gradients, and other theme properties.
