# Coraline Theme

Dark purple theme inspired by the Coraline movie.

## Assets Included

### banner.jpg
Banner image featuring characters from the Coraline movie (included).

**Specifications:**
- **Dimensions**: 1920x400px (wide aspect ratio)
- **Format**: JPG
- **Style**: Coraline movie imagery with button-eyed characters
- **Color palette**: Purples, blues, dark tones

## Theme Colors

- Primary: Purple (#5f4b8b)
- Accent: Light purple (#8a6cbd)
- Background: Very dark gray (#111827)
- Text: Light blue (#dbeafe)

## Customization

Edit `theme.json` to modify colors, gradients, and other theme properties.

You can replace `banner.jpg` with different Coraline imagery if desired, maintaining the same filename.
